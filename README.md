# Ryan Su | Portfolio

Personal portfolio website built with **Next.js 14**, React 18, Tailwind CSS, Framer Motion,
and Zustand. Features a scroll-driven hiking trail metaphor with animated SVG paths,
cinematic side-trail transitions, and a gamified progress HUD.

## Development

```bash
npm install
npm run dev       # http://localhost:3000
```

## Build

```bash
npm run build     # type-check + compile
npm start         # preview production build
```

## Lint

```bash
npm run lint
```

## Deploy

This is a Next.js app. Deploy to [Vercel](https://vercel.com) (recommended) or any
platform that supports Node.js / Next.js:

```bash
npm run build
npm start
```
